# Release Checklist

Target release: `v0.1.0 Public Preview`

## Required Before Publishing

- [ ] Privacy and customer material check completed.
  - No customer `.ai`, PDF, PPT/PPTX, screenshots, logos, brand assets, linked images, or source project files are present.
  - No private project names, client names, internal filenames, or hidden metadata are present in public docs.
- [ ] Absolute path check completed.
  - No local paths such as Windows drive paths, user home folders, machine usernames, or workspace paths remain.
  - Usage examples use relative paths or placeholders.
- [ ] Script syntax check completed.
  - `scripts/build_pptx.py` compiles.
  - `scripts/verify_pptx.py` compiles.
  - No `__pycache__` or generated bytecode remains in the repository.
- [ ] Example material check completed.
  - Any files added to `examples/` are fully self-made, desensitized, and publicly shareable.
  - `preview-source.png`, `preview-output.png`, `editable-demo.gif`, and `demo-output.pptx` contain no customer or third-party confidential material.
- [ ] README demo status checked.
  - README either includes public demo images/GIF, or clearly states that demo assets will be added later.
- [ ] Public GitHub page checked after publishing.
  - README renders correctly.
  - Links, headings, code blocks, and images/GIFs render as expected.
  - `.gitignore` did not hide required public files.
- [ ] Demo output validated in both WPS and PowerPoint.
  - `examples/demo-output.pptx` opens in WPS.
  - `examples/demo-output.pptx` opens in Microsoft PowerPoint.
  - Text boxes are editable.
  - Complex fonts, special effects, and individual line wrapping have been manually inspected.
