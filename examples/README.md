# Examples

This directory is intentionally empty except for this placeholder.

Do not commit customer files, source Illustrator documents, screenshots, logos, exported PNG layers, generated PPTX files, PDFs, or commercial fonts.

For `v0.1.0 Public Preview`, the repository may be published without demo assets. Later, add these four files only if they come from a fully self-made, desensitized, publicly shareable test file:

```text
examples/
  preview-source.png
  preview-output.png
  editable-demo.gif
  demo-output.pptx
```

- `preview-source.png`: screenshot or render of the self-made Illustrator source artboard.
- `preview-output.png`: screenshot of the generated PowerPoint/WPS result.
- `editable-demo.gif`: short screen recording showing that text boxes are editable.
- `demo-output.pptx`: generated PPTX demo file. This file is explicitly allowed by `.gitignore`.

Only add assets that you own or have explicit permission to publish. Do not include the source `.ai` file unless it is intentionally released under a public license.
